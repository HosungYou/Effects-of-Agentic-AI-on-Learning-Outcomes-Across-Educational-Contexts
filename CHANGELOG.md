## [0.1.0] - 2026-02-16

### Added
- Initial repository scaffolding migrated from dissertation_AI-adoption_meta
- Complete directory structure for standard meta-analysis (Hedges' g)
- 6-sheet coding template (Excel) for effect size extraction and AI agent coding
- 9 R analysis scripts (random-effects, moderator analysis, RVE, publication bias, HALO mapping)
- 7-Phase AI coding pipeline (Python) with Claude Sonnet 4.5 + GPT-4o + Groq consensus
- Comprehensive coding manual (Markdown + Word) for Agentic AI learning outcomes
- Search strategy documentation for 6 databases (WoS, Scopus, ERIC, PsycINFO, IEEE Xplore, ACM DL)
- Screening tools (AI-assisted screening, deduplication)
- Configuration files (pipeline config, analysis parameters)
- Supplementary materials (PRISMA 2020 checklist, preregistration protocol, codebook, RoB assessment)
- Master Integration Document for cross-module coordination
- HALO Framework specification (3-layer architecture)
- Data provenance tracking and quality gates
